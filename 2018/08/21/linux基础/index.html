linux基础

Posted on 2018-08-21 | Edited on 2018-08-23
Symbols count in article: 1.1k | Reading time ≈ 1 mins.
  • pwd:显示当前路径
  • clear 清屏
  • mkdir +文件夹 创建一个新文件夹
  • cd..:退回上一层目录
  • cd~:返回home目录
  • cd - :返回上一次路径
  • ls > xxx.txt 把应该显示在屏幕上的内容保存到xxx.txt
  • ls >> xxx.txt 追加
  • grep -n -v “要搜索内容” 文件 -n 行数 -v 不包含此搜索内容
  • sudo reboot 重启
  • touch 创建一个新文件
  • rm +文件名 删除文件
  • cat:显示或编辑文本文件
  • man+命令，查看某一个命令文档 /关键词 搜索功能 n 下一个 q 退出文档
  • linux中所有可读取设备都必须挂载之后才能使用，系统硬盘分区是linux的系统开机自动挂载的 所有的可读取设备都有一个设备文件名，挂载就是将设备文件名和挂载点(任意一个空目录)连接起来
  • ls:显示当前目录下文件或目录
  • ls / 显示指定文件夹下文件
  • ls -a 显示隐藏文件
  • ls -l 显示详细信息
  • 并且参数可以叠加 没有先后顺序
  • echo 命令用于在终端显示字符串或变量 格式 : echo [字符串/变量]
  • echo $SHELL 查看SHELL变量的值 (前面有$符号)
  • echo $HOSTNAME 查看主机名
  • date 查看系统时间
  • reboot 重启系统
  • wget 下载网络文件 wget[参数] 下载地址
  • -b 后台下载模式 -O 下载到指定目录 -t 最大尝试次数 -c 断点断传
  • -p 下载页面内所有资源，包括picture video -r 递归下载
  • elinks[参数] 网址 纯文本性质浏览器
  • ifconfig 获取网卡配置 与网络状态等信息 ifconfig [网络设备][参数]
  • uname 系统内核名称，版本,内核发行版，节点名，硬件名称，硬件平台，处理器类型。操作系统等信息 uname [-a]
  • uptime 查看系统负载情况 输出内容 系统当前时间 系统已运行时间 当前在线用户 平均负载值(分为最近一分钟 5 分钟 15 mins 系统负载情况)，越低越好 <1 正常
  • watch -n 1 update 每秒刷新一次获得当前系统的负载情况
  • free[-m/-g] 内存使用情况 m g 单位
  • who[参数] 用户情况
  • last [参数] 查看所有系统登入记录
  • history [-c]历史命令 会被保存到用户家目录.bash_history
  • -c 清空

  • sosreport 手机系统配置并诊断信息后，输出结论文档
