linux基础

Posted on 2018-08-21 | Edited on 2018-08-22
  • pwd:显示当前路径
  • clear 清屏
  • mkdir +文件夹 创建一个新文件夹
  • cd..:退回上一层目录
  • cd~:返回home目录
  • cd - :返回上一次路径
  • ls > xxx.txt 把应该显示在屏幕上的内容保存到xxx.txt
  • ls >> xxx.txt 追加
  • grep -n -v “要搜索内容” 文件 -n 行数 -v 不包含此搜索内容
  • sudo reboot 重启
  • touch 创建一个新文件
  • rm +文件名 删除文件
  • cat:显示或编辑文本文件
  • man 命令，查看文档
  • linux中所有可读取设备都必须挂载之后才能使用，系统硬盘分区是linux的系统开机自动挂载的 所有的可读取设备都有一个设备文件名，挂载就是将设备文件名和挂载点(任意一个空目录)连接起来
  • ls:显示当前目录下文件或目录
  • ls / 显示指定文件夹下文件
  • ls -a 显示隐藏文件
  • ls -l 显示详细信息
  • 并且参数可以叠加 没有先后顺序
